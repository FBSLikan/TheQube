h1. Что нового в TheQube

p. На этой странице представлены все изменения и исправления в TheQube, от более новых к более старым.

h2. TheQube 0.8

* Добавлено: Добавлен сервис сокращения ссылок "ux.nu":http://ux.nu.
* Изменено: Улучшена технология хранения данных учётной записи Твиттера.
* Исправлено: "#49: Для авторизации второй сессии требуется перезапуск TheQube":https://github.com/Oire/theqube/issues/49. Благодарим Мануэля Кортеса за это замечательное исправление!
* Исправлено: Переводчик Google снова с нами.
* Исправлено: ошибки в переводе на португальский (Бразилия), которые мешали TheQube нормально функционировать.

h2. TheQube 0.7.1

* Добавлено: Добавлен сервис сокращения ссылок "qps.ru":http://qps.ru.
* Исправлено: Временно удалена поддержка нерабочих сервисов сокращения ссылок, а именно onj.me и xed.cc, поскольку они теперь требуют ввода CAPTCHA.
* Исправлено: "#29: Файлы журнала вырастают до огромных размеров":https://github.com/Oire/theqube/issues/29. На данный момент максимальный размер файла установлен в 100 кб.
* Исправлено: снова можно отправлять аудиотвиты.
* Исправлено: нажатие клавиши *Enter* не добавляет новую строку при отправке твита.


h2.  TheQube 0.7

h3. Изменения

* Теперь Программа называется TheQube (без пробела) и подчиняется обычной системе нумерации версий _major.minor.patch_. До выхода первой стабильной версии первой цифрой номера будет 0, поэтому данная версия имеет номер 0.7.
* Вкладки «Речь» и «Брайль» в глобальной конфигурации программы объединены в одну, так как вкладка «Брайль» содержала только один флажок.
* Служба перевода твитов снова работает ("#27":https://github.com/Oire/TheQube/issues/27)
* Теперь поддерживаются аудиоссылки с изменённого сервиса AudioBoom.
* Теперь у TheQube есть постоянный многоязычный "сайт":http://theqube.oire.org
* Теперь в окнах создания сообщений работает комбинация клавиш *Ctrl+BackSpace*, удаляющая предыдущее слово.
* Теперь адрес web-сайта в диалоге информации о профиле не будет сокращаться с помощью _t.co_.
* Теперь в диалоге информации о профиле будет отображаться также число пользователей, добавивших текущего пользователя в свои списки.
* Папка _doc_ удалена из исходного кода, так как не содержит значимого руководства для разработчиков TheQube. Если штат разработчиков будет увеличен, такое руководство будет написано заново.
* Временно удалена поддержка сервиса сокращения ссылок hkc.im из-за ошибки сервиса, см. "их главную страницу":http://hkc.im/. Поддержку можно вернуть в любой момент, как только сервис заработает снова.
* Добавлена поддержка новых сервисов сокращения ссылок, а именно: _goo.gl_, _senta.la_ и _readability_
* При авторизации ввод пин-кода больше не требуется. Однако окно браузера всё же открывается, пока с этим сделать ничего нельзя ("#13":https://github.com/Oire/TheQube/issues/13)
* Обновлена локализация на испанский, сербский и польский языки. Спасибо нашей команде переводчиков!


h3. Исправления

* Исправлено: "#33: Переключение сессий прекращает работу клавиатурных команд":https://github.com/Oire/TheQube/issues/33
* Исправлено: "#1: Клавиатурные команды не работают, если по умолчанию раскладка клавиатуры не установлена на английский язык":https://github.com/Oire/TheQube/issues/1
* Исправлено: "#44: Язык документации TheQube не соответствует выбранному при установке":https://github.com/Oire/TheQube/issues/44]


h2. The Qube 1.0 beta 6

h3. Изменения

* сочетание клавиш для обновления буфера изменено на *Ctrl+Win+Alt+U*, чтобы разрешить конфликт с клавиатурными командами Windows 8.
* Исходный код The Qube перемещён на "Github":https://github.com/Oire/TheQube.
* Теперь у The Qube есть багтрекер — место, где можно сообщить о проблеме или предложить нововведение. Он находится "здесь":https://github.com/Oire/TheQube/issues. Сообщения лучше оставлять на английском языке.
* Теперь вы можете решить, отправлять сообщение по клавише *Enter* или по *Ctrl+Enter*. Соответствующий флажок находится в конфигурации The Qube, на вкладке «Разное». По умолчанию сообщения отправляются по клавише *Enter*, как и раньше.
* Теперь вы можете выбрать, следует ли загружать прикрепляемое аудио на "SndUp":http://sndup.net/ или на "TwUp":http://twup.me/.
* Сессия GoogleVoice удалена ввиду невозможности её тестирования разработчиками (сервис работает только в США).
* Теперь геолокация сообщается на том языке, который вы установили для интерфейса The Qube.
* сочетания клавиш для начала/прекращения чтения пользователя изменены на *Ctrl+Win+S*/*Ctrl+Shift+Win+S* для разрешения конфликта с функцией переключения языка в JAWS 16. Чтобы запомнить новые сочетания, используйте слова «Start/stop following».


h3. Исправления

* Теперь вы можете отправлять сообщения, содержащие переводы строки. По умолчанию перевод строки вставляется с помощью *Ctrl+Enter*. Однако вы можете пожелать отправлять сообщения по *Ctrl+Enter*, тогда перевод строки будет вставляться по клавише *Enter*.
* Редактирование сочетаний клавиш снова работает.
* Теперь вы можете выбрать голос SAPI5-синтезатора, если используете его для The Qube.
* Информация о геолокации снова сообщается всеми способами (*Ctrl+Win+G*, *Ctrl+Shift+Win+G*, *Ctrl+Shift+Win+Alt+G*).
